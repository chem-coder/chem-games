# pH Calculator

Migrated from `README.txt` on 2026-05-06.

## v1

Basic single-input, single-output program that takes the negative log of a concentration input to calculate and return pH.

## v2

Calculates pH of a solution of a strong or weak acid or base.

Current behavior:

- Requires both concentration and dissociation constant for weak acid or weak base calculations.
- Prevents users from entering values in multiple calculation categories at once.
- Calculates pH from one input group only.
